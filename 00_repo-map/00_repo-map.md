# AI Foundations | Agent Execution Boundary
# Repo Map

## Purpose

This repository defines the public boundary between autonomous agent execution and source authority.

It establishes a proof layer for one core claim:

Execution capacity is not source authority.

An agent may execute inside a frame.

The agent does not become the source of the frame.

## Repository Function

This repository is not an agent implementation.

It is not a private architecture dump.

It is not a full protocol release.

It is a public-safe framework layer naming the authority, provenance, approval, and validity boundaries that must exist before autonomous agents act.
