# Output Is Not Provenance

## Definition

Output is what a system produces.

Provenance is the source record of origin, authorship, structure, authority, and release.

Output is not provenance.

An agent may produce output inside a frame.

The agent does not become the source of the frame by producing output.

## Core Claim

Produced output does not create provenance.

A system may generate text.

A system may summarize a framework.

A system may create a diagram.

A system may write a policy.

A system may produce a workflow.

A system may generate a strong answer.

None of these outputs prove origin.

Output can express a frame.

Output can describe a frame.

Output can assist a frame.

Output does not become the source record of the frame.

## Why This Distinction Matters

Autonomous agents can produce fluent, useful, and polished outputs at scale.

Because the output exists, people may treat the output as if it proves source.

That is the failure.

A clean answer is not provenance.

A useful summary is not provenance.

A working implementation is not provenance.

A generated document is not provenance.

A derivative output is not provenance.

Output is not the same as origin.
