# Agent-Generated Summary Is Not Authorship

## Definition

Agent-generated summary is output produced by an agent that describes, condenses, explains, reorganizes, or translates an authored work.

Authorship is the source-position of origin, structure, selection, decision, release, and accountability.

Agent-generated summary is not authorship.

A summary may describe the framework.

The summary does not become the source of the framework.

## Core Claim

A summary does not create authorship.

An agent may summarize a framework.

An agent may explain a framework.

An agent may reorganize a framework.

An agent may translate a framework.

An agent may generate public-facing language from a framework.

None of these outputs make the agent the author.

Output is not provenance.

Summary is not authorship.

Use does not transfer Origin.
